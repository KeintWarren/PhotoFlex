PhotoFlex

PhotoFlex is an interactive image board web application that allows users to create personalized boards and organize pins (images) in a visually engaging way. Similar to Pinterest, it enables users to explore, save, and interact with content while customizing the platform to their own style.

Features

Custom Boards: Create and manage multiple boards to organize pins by themes or interests.

Pin Management: Add images as pins, like, comment, and save content to your boards.

User Interaction: Engage with other users through comments, likes, and saved pins.

Customization: Personalize the website’s appearance to match your style.

Explore & Discover: Browse and find inspiration from other users’ boards and pins.

Technology Stack

Frontend:

React (JSX)

HTML5

CSS3

Backend:

Spring Boot (Java)

Database: MySQL or PostgreSQL

API: RESTful services for interaction between frontend and backend
